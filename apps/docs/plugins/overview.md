# Plugins Overview

## Official Plugins

| Plugin | Description |
|--------|-------------|
| `@hanabi/plugin-auth` | Authentication & session management |
| `@hanabi/plugin-blog` | Blog posts & categories |
| `@hanabi/plugin-media` | File upload & media library |

## Install

```bash
bun add @hanabi/plugin-blog
```

## Community Plugins

Community plugins can be installed from the Admin UI Plugin Marketplace. They are loaded dynamically from CDN without requiring a server restart.
