# @viseed/registry

Plugin registry client for [Viseed CMS](https://viseed.github.io/cms). Provides `PluginRegistryClient` for fetching the plugin list, installing and uninstalling plugins, and verifying bundle SHA-256 integrity.

## Install

```sh
bun add @viseed/registry
```

> Included transitively via `@viseed/cms`.
