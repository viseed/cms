# @viseed/types

TypeScript type definitions for [Viseed CMS](https://viseed.github.io/cms). Contains all shared interfaces and types — `CMSPlugin`, `CMSTheme`, `CMSConfig`, `CMSPluginHooks`, and more. No runtime code.

## Install

```sh
bun add -d @viseed/types
```
