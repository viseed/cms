# @viseed/ui

Vue 3 component library for [Viseed CMS](https://viseed.github.io/cms). Exports `ComponentRegistry`, `themeWidgetRegistry`, base UI components, the Plugin Marketplace view, and a `RichTextEditor` (TipTap-based) via the `@viseed/ui/vue` subpath.

## Install

```sh
bun add @viseed/ui
```

> Consumed by `apps/admin` and official plugins. Not typically installed by end users directly.
