# @viseed/validator

Zod validation schemas for [Viseed CMS](https://viseed.github.io/cms). Covers content, users, media, and plugin payloads. Used by `@viseed/core` and official plugins to validate API input.

## Install

```sh
bun add @viseed/validator
```

> Included transitively via `@viseed/cms`. Install directly only when building plugins or custom route handlers.
